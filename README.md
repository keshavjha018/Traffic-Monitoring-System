### Please follow the given steps for installation

* Step:1 - Install virtualenv & activate it.
  ```sh
  pip install virtualenv
  ```

* Step:2 - Install requirements & Flask.
  ```sh
  pip install -r requirements.txt
  ```
   ```sh
  pip install Flask
  ```

* Step:3 - Run Command
  ```sh
  Flask run
  ```
